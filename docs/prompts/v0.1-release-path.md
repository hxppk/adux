## Style guardrails（与 usage-guide 现有 5 张图保持一致）

- 风格：**flat illustration + 轻微 isometric / 浅景深**，柔和 vector 描边
- 背景：**off-white #FAFAFA → 浅蓝灰 #F0F4FA 的极轻渐变**，留白充足
- 主色板：
  - 主色 `#1677FF`（Ant Design 蓝，用于"已完成 / 当前"高亮）
  - 阶段块底色 `#E6F4FF`（已完成）、`#FFF7E6`（当前）、`#F5F5F5`（待完成）、`#F6FFED`（终点目标）
  - 文字深灰 `#262626`，副文字 `#8C8C8C`
  - 警示橙 `#FA8C16`（用于 blocker 标签）
  - 成功绿 `#52C41A`（用于 ✓ 标记）
- 字体：sans-serif（Inter / PingFang SC 视觉感），加粗用于阶段标题
- 图标风格：**rounded square outline + 1.5px stroke + 极简 fill**，与 antd 官方插图一致
- 不要 AI 拟物风、不要照片合成、不要中文模糊渲染——文字一律清晰可读

## Composition

横向四阶段，从左到右流动，阶段间用 `→` 粗箭头连接（箭头颜色 `#1677FF` 50% 透明）。每个阶段一个**圆角矩形卡片**（圆角 24px，阴影 0 4px 12px rgba(0,0,0,0.06)），上半区放阶段图标 + 阶段标题 + 版本号，下半区放 2-3 行说明 + 状态标签。

底部预留一条「Blockers」时间线，把 R10/R6/CI smoke 等关卡用小药丸贴在对应阶段下方。
